## 1. 프로젝트 개요

### 1.1 목적
대전 지역 전기 자격증 전문 교육기관인 **플러스 전기학원**의 공식 홈페이지를 제작한다. 인천대산전기직업학교 사이트를 레퍼런스로 하되, 학원 정체성에 맞춰 콘텐츠를 재구성하고 모던하고 고품질의 상용 버전으로 제작한다.

### 1.2 비즈니스 목표
- 잠재 수강생 유입 및 문의 전환
- 전기 자격증 과정 정보 제공 (전기기능사, 전기기사, 전기산업기사 등)
- 학원 신뢰도 및 브랜드 이미지 구축
- 모바일/PC 모두에서 우수한 사용자 경험 제공

## 5. 비기능 요구사항 (Non-Functional Requirements)

### 5.1 성능
- Lighthouse Performance ≥ 90 (모바일)
- LCP ≤ 2.5s, CLS ≤ 0.1, INP ≤ 200ms
- 이미지 최적화 (WebP, 반응형 srcset, lazy loading)

### 5.2 SEO
- 페이지별 `<title>`, `meta description` 최적화
- Open Graph, Twitter Card 메타태그
- `sitemap.xml`, `robots.txt`
- 구조화 데이터 (JSON-LD): `EducationalOrganization`, `LocalBusiness`
- 한국어 우선 (`<html lang="ko">`)

### 5.3 접근성 (a11y)
- WCAG 2.1 AA 준수
- 키보드 네비게이션 가능
- 색상 대비 4.5:1 이상
- 이미지 `alt` 속성 필수
- 시맨틱 HTML (`<header>`, `<main>`, `<nav>`, `<footer>` 등)

### 5.4 브라우저 지원
- 최신 Chrome, Safari, Firefox, Edge (최근 2개 버전)
- iOS Safari 15+, Android Chrome 최신

### 5.5 보안
- HTTPS 필수 (Cloudflare Pages 자동 제공)
- 외부 링크는 `rel="noopener noreferrer"`
- 폼 제출 시 reCAPTCHA 또는 honeypot (확장)

## 6. 디자인 가이드라인

### 6.1 디자인 톤 & 무드
- **신뢰감 있는 전문 교육기관** 이미지 (보수적이지 않고, 모던하면서 신뢰감)
- 깔끔하고 정돈된 그리드 레이아웃
- 여백을 충분히 활용한 모던 디자인 (원본 사이트보다 한층 업그레이드된 비주얼)

### 6.2 컬러 팔레트
원본 사이트의 딥 블루(#363fa0)를 베이스로 모던하게 재해석.

| 역할 | 컬러 | HEX |
|------|------|-----|
| Primary (브랜드 메인) | Electric Blue | `#1E40AF` |
| Primary Dark | Deep Navy | `#1E3A8A` |
| Accent (포인트) | Vivid Yellow | `#FACC15` |
| Success / 합격 | Green | `#10B981` |
| Background | White | `#FFFFFF` |
| Surface (카드) | Soft Gray | `#F8FAFC` |
| Text Primary | Slate-900 | `#0F172A` |
| Text Secondary | Slate-600 | `#475569` |
| Border | Slate-200 | `#E2E8F0` |

> **컨셉**: "Plus(+)" → 더한다, 충전한다, 전기(electric) → 전기 블루 + 노란 액센트로 전기/에너지의 시각적 메타포 강조.

### 6.3 타이포그래피
- 한글: **Pretendard** (또는 Noto Sans KR)
- 숫자/영문: **Inter**
- 헤딩: 700 (Bold), 본문: 400~500
- 헤딩 스케일: H1 48px / H2 36px / H3 28px / H4 22px / Body 16px

### 6.4 컴포넌트 스타일
- Border-radius: 12px (카드), 8px (버튼)
- 그림자: 2단계 (`shadow-sm`, `shadow-lg`)
- 트랜지션: 150~250ms ease-out
- 포커스 링: Primary 컬러 2px outline

### 6.5 로고
- MVP 단계: 워드마크 형식 ("**플러스 전기학원**" + ⚡ 아이콘)
- SVG로 제작하여 무한 확장 가능

## 7. 기술 스택

### 7.1 선정 기준
- Cloudflare Pages 배포 호환
- 정적 사이트로 빠른 로딩
- 모던 개발 경험 (TypeScript, 컴포넌트 기반)
- 향후 게시판 등 동적 기능 확장 가능

### 7.2 스택

| 영역 | 기술 | 비고 |
|------|------|------|
| 프레임워크 | **Next.js 15 (App Router)** | 정적 export 또는 Cloudflare adapter |
| 언어 | **TypeScript** | 타입 안정성 |
| 스타일링 | **Tailwind CSS v4** | 유틸리티 퍼스트 |
| UI 컴포넌트 | **shadcn/ui** | 필요한 만큼만 |
| 아이콘 | **lucide-react** | 모던 아이콘 |
| 폰트 | **Pretendard (CDN)** | 한국어 최적화 |
| 애니메이션 | **Framer Motion** (선택) | 부드러운 전환 |
| 폼 처리 | **Web3Forms / mailto** | 서버리스 |
| 분석 | Cloudflare Web Analytics | 프라이버시 친화 |
| 배포 | **Cloudflare Pages** | GitHub 연동 자동 배포 |
| 호스팅/CDN | Cloudflare 글로벌 CDN | 자동 |

### 7.3 정적 export 방식
- Next.js `output: 'export'` 사용 → Cloudflare Pages에 정적 자산으로 배포
- 폼 등 서버 기능이 필요해지면 Cloudflare Pages Functions로 점진 전환
